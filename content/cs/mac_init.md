# 电脑安装指南

## 电脑基础文件配置
```shell
#!/bin/bash

# 创建主文件夹
mkdir ~/box

# 创建子文件夹
mkdir ~/box/1_code
mkdir ~/box/2_doc
mkdir ~/box/3_file
mkdir ~/box/4_config

# 在1_code文件夹中创建子文件夹
mkdir ~/box/1_code/1_frontend
mkdir ~/box/1_code/2_backend
mkdir ~/box/1_code/3_client
mkdir ~/box/1_code/4_script

mkdir ~/box/4_config/maven_setting
```
